# Exemplos do CLINUX

Esta pasta contém exemplos práticos de uso do CLINUX via Python.

## Scripts

- `create_ubuntu_vm.py` - Criar VM Ubuntu
- `quick_proot.py` - Ambiente PRoot rápido
- `snapshot_workflow.py` - Gerenciar snapshots

## Uso

```bash
python create_ubuntu_vm.py
python quick_proot.py
python snapshot_workflow.py
```
