# Changelog

## [1.0.0] - 2024-01-01
### Added
- Comando `install` para instalar ISO com QEMU
- Comando `run` para iniciar VMs
- Comando `proot` para ambientes rootfs
- Comando `list` para listar recursos
- Comando `delete` para remover VMs
- Comando `snapshot save/load` para snapshots
- Suporte a discos qcow2
- Suporte a rootfs tar.gz e diretórios
- Detecção automática de dependências

### Fixed
- Correção de bugs na criação de discos
- Melhor tratamento de erros

## [0.1.0] - 2023-12-01
### Added
- Versão inicial com comandos básicos
